# Scenario Context

Launching AgentHub in Q2–Q3 2026 as Docebo's largest release to date. This moves
Docebo beyond the traditional LMS category into a unified platform combining
learning, enterprise knowledge, and skills intelligence with agentic AI. AI
agents are coming this fall.

## Acquisition foundations
- **Zive acquisition** provides the enterprise knowledge layer.
- **365Talents acquisition** provides skills intelligence.

## Competitive context
- Competing against legacy LMS players: Cornerstone OnDemand, SAP SuccessFactors,
  Workday Learning — while also defining a new category that doesn't exist yet.
- Competing against standalone knowledge management tools: Notion, Guru,
  Bloomfire.
- Competing against skills intelligence platforms: Gloat, Eightfold.

## Differentiation
Having all three — learning + knowledge + skills — in one closed loop with
agentic AI.
